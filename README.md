# nodejs-cli-crypto-pbkdf2-encoded

## Description
A bases for encoding a password
with pbkdf2.

## Tech stack
- nodejs

## Docker stack
- node:latest

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
